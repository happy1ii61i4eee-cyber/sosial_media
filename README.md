# 🗨️ Social Media Platform
一個使用 Node.js + Express + PostgreSQL 建構的簡易社交平台。

## 🚀 功能
- 使用者註冊 / 登入 (JWT 驗證)
- 發文 / 留言 / 搜尋
- 會員資料頁面
- RESTful API 架構

## 🛠️ 技術棧
- Node.js / Express
- PostgreSQL / bcrypt / JWT
- EJS + Fetch API
- Docker (可選)
